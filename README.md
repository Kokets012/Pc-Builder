# PC Builder Compatibility Checker

A web application that helps users build compatible PC configurations with real-time compatibility checking and price calculation in South African Rands.

## Features

- Component selection (CPU, Motherboard, RAM, GPU)
- Real-time compatibility validation
- Price calculation in ZAR
- Save and manage builds
- Delete functionality

## Installation

1. Clone repository
2. Rename `config.example.php` to `config.php`
3. Update database credentials in `config.php`
4. Import `database.sql` to your MySQL database
5. Deploy to your web server

## Technologies

- PHP (Procedural)
- MySQL
- Bootstrap 5
- JavaScript

Import the provided SQL file to set up the database structure and sample components.
